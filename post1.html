Wallpaper App

This was a project I started back in november 2020 when I first learned Flutter.

This is a fun interactive Ios and Android app.User has access to thousands of images fetched from pexels.

User can sruf through images,search for images and save them in the mobile's internal storage for free.

The application is developed with flutter which is a free and open-source mobile UI framework created by Google and released in May 2017. In a few words, it allows you to create a native mobile application with only one codebase.

Technologies:
  • - Flutter
  • - Dart
  • - Pexels API
